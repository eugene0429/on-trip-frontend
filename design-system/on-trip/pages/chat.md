# Page Override: Chat (S-10 1:1 채팅방)

## 레이아웃

```
┌─────────────────────────────┐
│ [SafeArea]                  │
│ ←  [사진]@닉네임      ⋯    │  ← 좌:back, 중:상대 mini, 우:신고/차단/나가기
│       광안리 일행            │     상태: 활성/만료
├─────────────────────────────┤
│  [안전 안내 토스트 카드]    │  ← 첫 진입 1회: "외부 연락처는 자동으로 가려져요"
├─────────────────────────────┤
│                             │
│  [상대]  안녕하세요!        │
│         오후 3:14  · 읽음   │
│                             │
│         반가워요 ㅎㅎ  [나]│
│                  오후 3:15  │
│                             │
│  [상대]  광안리 어디 계세요? │
│         혹시 010-***-****   │  ← 자동 마스킹된 메시지
│         오후 3:18  · 읽음   │
│                             │
├─────────────────────────────┤
│  ┌─────────────────────┐    │
│  │ 메시지 입력...    [📷]│    │  ← StickerInput full-width + 전송 아이콘
│  │                  [→] │    │     이미지(V2)·전송
│  └─────────────────────┘    │
│  [마스킹 안내 토스트 1회]   │  ← 키워드 감지 시 발신자 화면에만 표시
└─────────────────────────────┘
```

## 헤더

- 좌측: ← (Lucide arrow-left)
- 중앙: 32dp 사진 원형 + outline + 닉네임 title + "광안리 일행" caption.
- 우측: ⋯ → 액션시트 ([신고] [차단] [채팅 나가기])
- 헤더 하단 outline 1.5 보더 (스크롤 시에도 그림자 X).
- 활성 상태: 닉네임 옆 라임 점.
- 만료(상대 Trip 만료): "여행이 끝났어요" 회색 칩 + 메시지 입력 가능 유지.

## 메시지 버블

### 내 메시지 (오른쪽 정렬)

- 배경: `primary` 노랑 (#FFE066) + outline 2.5
- 텍스트: outline 검정
- radii: 20dp + 우하단 8dp (꼬리 효과)
- max width: 화면 너비의 75%
- 그림자: shadow.sm (4/4)
- 시각·읽음 표시: 버블 아래 caption textMuted

### 상대 메시지 (왼쪽 정렬)

- 배경: surface 흰색 + outline 2.5
- 텍스트: outline 검정
- radii: 20dp + 좌하단 8dp
- 사진 32dp는 첫 메시지에만 (연속 메시지는 생략 — 카톡 패턴)
- 그림자: shadow.sm

### 시스템 메시지 (가운데 정렬)

매칭 시작/만료/차단 등.

- 작은 회색 칩 (radii.pill) + caption + outline 1.5
- 예: "✨ 매칭 시작됨", "⚠️ 상대가 채팅방을 나갔어요"

### 마스킹된 메시지

- 마스킹 부분만 진한 회색 박스 + "***" 텍스트.
- 발신자 화면: 원본 텍스트 그대로.
- 첫 마스킹 발생 시 발신자에게 1회 토스트 "외부 연락처는 가려져 전달됐어요 🛡️" — FR-042.

## 입력 영역

- 위치: KeyboardAvoidingView로 키보드 위 고정 + safe-area-inset-bottom.
- StickerInput multiline (1~4줄 자동 확장). placeholder: "메시지 입력...".
- 우측 아이콘: 📷 카메라(V2 회색 disabled) + → 전송 (텍스트 있을 때만 노랑 활성).
- 전송 활성 시 mechanical press + impactLight (햅틱 — 가장 약함).

## 인터랙션

- **메시지 길게 누름:** 액션시트 [복사] [신고]. 신고는 해당 메시지 ID 첨부.
- **이미지 메시지 (V2):** 탭 시 lightbox.
- **읽음 표시:** 상대가 채팅방 진입 시 socket으로 갱신. 내 메시지 옆 "읽음" 회색 caption.
- **타이핑 인디케이터:** 상대 입력 시 좌하단 "...•••" 애니메이션 (V2, MVP는 생략 가능).
- **스크롤 to bottom 버튼:** 스크롤 위로 올라간 상태에서 새 메시지 도착 시 우하단 작은 노랑 원형 버튼 등장.

## 안전 기능 UX (FR-041~043)

- **첫 진입 토스트(영구):** "이 채팅은 1:1 비공개예요. 외부 연락처(전화번호·SNS·계좌)는 자동으로 가려져요." (라임 배경, 5초 후 사라짐, 채팅방 정보에서 다시 볼 수 있음)
- **차단 동작:** 차단 확정 시 즉시 채팅방 종료 → 활동 탭으로 돌아감 + 시스템 메시지 "차단된 채팅이에요"는 양쪽 다 보지 못함.
- **신고 동작:** 신고 모달(S-12) 진입 → 5종 사유 + 50자 상세 → 제출 시 서버 처리. 시스템 메시지 노출 X (상대가 신고 사실 모름).

## 상태

- **메시지 0건:** 빈 영역 + "첫 메시지를 보내보세요!" + 큰 일러스트.
- **상대 차단:** "이 채팅을 더 이상 볼 수 없어요" 풀스크린 + [돌아가기].
- **네트워크 끊김:** 상단 작은 빨강 라인 + "오프라인" 텍스트 + 재연결 시 자동 메시지 동기화.
- **메시지 전송 실패:** 메시지 우측에 빨강 ! 아이콘 + 탭 시 [재전송] [삭제].

## 안티패턴

- ❌ 메시지 버블에 회전 (가독성)
- ❌ 그림자 큼 (메시지가 많아지면 시각 노이즈, sm만)
- ❌ 마스킹된 부분을 발신자에게도 가림 (FR-042 위반)
- ❌ 첫 진입 토스트 미노출 (안전 메시지는 필수)
- ❌ 채팅방 외부에서 알림 미노출 (FCM 푸시 필수)
