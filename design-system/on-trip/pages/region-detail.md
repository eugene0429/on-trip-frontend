# Page Override: Region Detail (S-04 거점 상세)

## 레이아웃

```
┌─────────────────────────────┐
│ [SafeArea]                  │
│ ←  광안리              ⋯    │  ← 좌: back, 중: 거점명+인구칩, 우: 더보기
│       (24 명) ●활성         │     "활성" 칩은 라임색
├─────────────────────────────┤
│  [필터 칩 가로 스크롤]      │
│  [전체] [맛집] [술자리]    │  ← Chip pill, 선택 칩은 노랑 배경 + outline
│  [액티비티] [관광] [사진]  │
├─────────────────────────────┤
│  ┌─────────────────────┐    │
│  │ [사진]  @닉네임    │    │  ← StickerCard (offset 6)
│  │         · ENFP     │    │     일행 카드 1
│  │ ──────────────     │    │
│  │ 일행: 남2 여1 · 20후│    │
│  │ #맛집 #술자리      │    │
│  │ "광안리 맛집 ..."   │    │

│  │  │ 콕 찌르기 100🪙│ │    │  ← StickerButton primary, full-width

│  └─────────────────────┘    │

│      ...                    │
└─────────────────────────────┘
```

## 핵심 사양

- **리스트:** `@shopify/flash-list` + `estimatedItemSize: 220`. 카드 사이 vertical gap 16dp.
- **헤더:** sticky, 스크롤 시 그림자 그라데이션 X (대신 outline 1.5 보더).
- **인구 칩:** "(24명) ●활성" — `accentLime` 점 + 카운트 (JetBrains Mono).
- **필터 칩:** 가로 스크롤, 활성 1개만 (단일 선택). 컨셉 6종 + "전체".
- **카드 회전:** **전체 카드의 30%만** 결정론적 ±1deg (id 해시 mod 3 == 0). 나머지는 0deg. → 시각적 활기 + 가독성 균형.

## 일행 카드 구성

```
[프로필 사진 64dp + outline] [@닉네임 title] [· MBTI caption]
                            [굵은 outline 라인 1.5]
[일행 정보 body: "남2 여1 · 20후"]
[Chip 가로 스크롤: 컨셉 태그]
["일행 한줄소개" body, max 2줄, ellipsis]
[StickerButton primary: "콕 찌르기 [100🪙 코인배지]"]
```

- 사진 없는 일행: 노랑 원형 배경 + outline + 닉네임 첫 글자(display).
- 컨셉 칩: 5개 초과 시 가로 스크롤 (overflow visible).
- 일행 한줄소개: 50자 제한이지만 카드에선 2줄 truncation, 탭하면 펼쳐짐.

## 인터랙션

- **카드 탭:** mechanical press → 일행 프로필(S-05)로 navigate.
- **콕 버튼 탭:** 카드 내부 터치 이벤트 stop propagation. 코인 부족 시 비활성 + 안내 토스트.
- **필터 칩 탭:** selection 햅틱. 리스트 즉시 필터 (서버 재요청 X, 클라이언트 필터).
- **스크롤:** scrollEventThrottle 16, 스크롤 위치 상태 보존(navigation 복귀 시).

## 상태

- **로딩:** 카드 모양의 skeleton 4개 (StickerCard 그림자 미적용 회색 박스).
- **0건:** 큰 SVG 스티커 + "이 거점엔 아직 일행이 없어요" + [내 일행 등록하기] CTA.
- **에러:** 우측 상단 작은 배지 "재시도" + 라스트 굿 데이터 유지.

## 안티패턴

- ❌ 카드 모두 회전 (30%만)
- ❌ 콕 버튼 작게 (반드시 카드 내부 메인 CTA, 56dp 미만 X)
- ❌ 필터 칩 라디오 모양 (디자인 일관성 — pill 형태 유지)
- ❌ "보낸 콕"임을 카드에 안 표시 (이미 콕 보낸 일행은 카드 우상단에 작은 노랑 배지 "콕 보냄" 표시)
